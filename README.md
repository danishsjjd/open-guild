# OpenGuild

OpenGuild is a Next.js app for building an open-source team and community chat experience inspired by modern guild, workspace, and channel-based communication tools.

## Getting Started

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
pnpm format
pnpm format:check
```
